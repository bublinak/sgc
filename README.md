# sgc

Studentuv pruvodce Kybernetikou.

Tento repozitar hostuje Docsify web pripraveny pro GitHub Pages. **[here](https://kybernetika-spse.github.io/sgc/)**

## Publikace

- GitHub Pages source: `main` / `docs`
- vstupni soubor webu: `docs/index.html`
- hlavni obsah: markdown soubory v `docs/`

## Struktura webu

- `docs/README.md` - hlavni stranka
- `docs/aktualni-informace/README.md` - aktualni informace o predmetu
- `docs/seznam-maturitnich-otazek/README.md` - hlavni seznam temat
- `docs/mo-jirka/README.md` - Jirkovy materialy
- `docs/mo-pavel/README.md` - Pavlovy materialy
- `docs/nova-skripta/README.md` - nova skripta

## Lokalni nahled

Staci otevrit obsah slozky `docs/` pres jednoduchy staticky server nebo po nastaveni GitHub Pages pouzit publikovanou URL repozitare.
